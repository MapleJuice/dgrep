# dgrep
Distributed grep

# Install from Release
```bash
./configure
make
sudo make install
```

# Install from Source
	Install <href = "https://www.gnu.org/software/autoconf/autoconf.html" target = "_blank">Autoconf</a> and <href = "https://www.gnu.org/software/automake/" target = "_blank">Automake</a>, then run

```bash
autoreconf --install
./configure
make
sudo make install
```
